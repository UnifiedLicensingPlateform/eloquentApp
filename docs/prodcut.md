Eloquent App - Zero-Cost MVP Plan
Product Vision
AI-powered speech coach that reduces word repetition through real-time feedback and practice drills.

Core Features (MVP)
Practice Drills - 60-second speaking exercises with transcription

Word Repetition Detection - Highlights repeated words in transcripts

Progress Dashboard - Tracks fluency scores and repetition patterns

User Profiles - Save practice history and metrics

Tech Stack (Zero-Cost)
Frontend: React + Vite deployed on Netlify (free tier)

Backend: Supabase (free tier) for auth, database, and storage

Speech-to-Text: Web Speech API (browser-native, free)

UI: shadcn/ui (free, open-source)

Implementation Phases
Phase 1 (Weeks 1-2)
Basic recording interface with Web Speech API

Word repetition detection algorithm

Supabase auth and user profiles

Phase 2 (Weeks 3-4)
Progress dashboard with charts

Practice history and analytics

Enhanced transcription accuracy

Key Metrics
Activation: Users completing first drill

Engagement: Drills per week per user

Success: Reduction in repetition rate over 30 days

Cost Structure
Netlify: 100GB bandwidth, 300 build mins/month (free)

Supabase: 500MB database, 50K users, 1GB storage (free)

Total Project Cost: $0
